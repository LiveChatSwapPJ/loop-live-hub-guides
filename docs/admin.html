管理者向け手順書

初期設定、配信者追加、写真フェイススワップ、利用料金確認、ユーザー情報変更の手順です。

1. 初期設定

目標: 管理者自身のアカウント作成と、有効化待ちまで完了する。

1アクセスとアカウント作成タブ

操作LoopLive にアクセスし、アカウント作成 タブを選択

2利用規約の同意

操作利用規約をスクロールし、同意する を押す

3Email / パスワード入力〜作成

操作Email と パスワード を入力し アカウントを作る を押す

運用有効化待ち

アカウント作成後は、サービス提供元に連絡し、有効化されるまで待ってください。

2. 配信者を追加する

目標: 配信者のアカウント作成後、管理者が権限と顔画像設定を反映する。

前提配信者側のアカウント作成

配信者には「1. 初期設定」の手順（アカウント作成）を実施してもらってください。

1ユーザー管理を開く

操作ユーザーを管理する を押す

2対象配信者を選択して編集

操作対象カードをクリックし、編集 を押す

3権限・顔画像設定と更新

操作権限（権限の一覧）と顔画像（source_file）を設定し、更新 を押す

3. 写真フェイススワップ

目標: 写真フェイススワップを実行し、結果をダウンロードする。

1画像2枚を選択

操作元の顔を選択 と なりたい顔を選択 で画像をアップロード

2フェイススワップを実行

操作フェイススワップする → 確認ダイアログで はい

3完了待ちとダウンロード

操作処理履歴が 完了 になるまで待ち、結果 をダウンロード

4. 利用料金を確認する

目標: 利用時間・請求金額・利用履歴・フォトスワップ使用枚数を確認する。

1利用料金画面へ移動

操作サーバー使用料を見る を押す

2合計値と明細を確認

操作合計利用時間 / 合計請求金額、明細行を確認

3利用履歴とフォトスワップ使用枚数

操作利用履歴ダイアログと フォトスワップ使用 を確認

5. ユーザー情報を変更する

目標: ユーザーの検索・編集・削除を行う。

1ユーザー管理を開く

操作ユーザーを管理する を押す

2検索して対象ユーザーを選択

操作検索欄に条件を入力し、対象カードを選択

3編集して更新

操作編集 → 必要項目（名前/Email/パスワード/権限）更新 → 更新

4削除

操作削除 を押し、確認ダイアログで はい
