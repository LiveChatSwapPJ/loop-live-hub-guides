ユーザー向け手順書

アカウント作成方法とリアルタイム加工の操作手順です。

1. アカウントを作る

目標: ユーザーが自身のアカウントを作成できる。

1アクセスとアカウント作成タブ

操作LoopLive にアクセスし、アカウント作成 タブを選択

2利用規約をスクロール

操作利用規約を最下部までスクロール

結果確認同意する が有効化される

3同意する

操作同意する を押す

4Email / パスワード入力

操作Email と パスワード を入力

5アカウントを作る

操作アカウントを作る を押す

2. リアルタイム加工

目標: 加工開始から加工停止まで実行し、必要に応じて加工後映像を確認する。

1ポータル — 加工を始める

操作加工を始める を押す

2サーバー管理 — 入室

操作対象サーバーで 入室 を押す

3起動と準備完了の確認

操作必要時は 起動する → 確認ダイアログで はい、準備完了 まで待機

4カメラ接続〜加工開始

操作カメラに接続 後、必要ならカメラ/鼻下設定を調整し 加工開始

5加工処理中〜停止

操作確認ダイアログで はい を押し、加工処理中… → 加工停止 まで待つ

6加工後映像確認と停止

操作必要に応じて 加工後の映像を見る、終了時は 加工停止

補足（所要時間）

  • 起動の確認〜準備完了（手順4〜5）は、数分〜十数分かかる場合があります。
  • 加工の確認〜加工処理中（手順10〜11）は、数十秒〜数分かかる場合があります。
